# Council-Of-LLMs

## How to Use

1. Have Ollama and models installed. I'm using Arch based CachyOS
curl -fsSL https://ollama.com/install.sh | sh
ollama run <insert-model-name-here>

2. Make council.py executable
chmod +x council.py
